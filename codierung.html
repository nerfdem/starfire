  • Vorreiter: Morse & Lochkarten

    Lange vor dem PC nutzte man Codes: Morsezeichen für Telegrafen und Lochkarten für Webstühle.

    • Lochkarte: Loch = 1, kein Loch = 0.
    • Erste mechanische Datenspeicherung.
    • Prinzip: Einem Muster wird eine Bedeutung zugeordnet.

  • ASCII: Der erste Standard

    In den 1960ern einigte man sich auf einen 7-Bit-Code für Text (128 Zeichen).

    • Reichte für Englisch, Zahlen und Steuerzeichen.
    • Problem: Keine Umlaute (ä, ö, ü) oder asiatische Schrift.
    • Führte oft zu "Datensalat" beim Austausch.

  • Unicode: Sprache für alle

    Unicode gibt jedem Zeichen der Welt – von Arabisch bis Emoji – eine eindeutige Nummer.

    • Löst das ASCII-Problem global.
    • UTF-8 speichert diese Nummern platzsparend.
    • Standard im heutigen Web für fehlerfreie Darstellung.

  • Farbcodes: RGB & Hex

    Auch Bilder sind Codes: Jeder Pixel wird aus Rot, Grün und Blau (RGB) gemischt.

    • Jeder Farbkanal hat einen Wert von 0 bis 255.
    • Hex-Code (z.B. #FF0000) ist verkürzte Binärschreibweise.
    • Ein Bild ist nur eine Matrix aus Zahlenwerten.

  • Fehlerkorrektur

    Bei der Übertragung können Bits "umkippen" (aus 0 wird 1). Codes schützen davor.

    • Prüfsummen oder Paritätsbits erkennen Fehler.
    • System kann defekte Daten oft selbst reparieren.
    • Unverzichtbar für Streaming und Downloads.
